Pour utiliser le générateur de playlist il faut :

*copier les fichiers MatrixScore.py, FindMusic.py, MusicLauncher.py dans le répertoire où se trouvent l'ensemble des musiques. 

*lancer le script python avec la commande python3 MusicLauncher.py (python3 est requis pour l'exécution).

*Un  lecteur est alors censé s'afficher. Les deux premières chansons sont choisies aléatoirement, ensuite le choix des chansons se fait selon la matrice des scores.

*La matrice des scores est crée si elle n'est pas trouvée dans le dossier courant. Ne pas la modifier.

*Ne pas tenir compte des messages d'erreurs de vlc.py apparaissant en rouge dans le terminal.
